# Data Model

## Public repo

This public repo carries:
- generic skills
- generic tooling
- markdown docs
- minimal example data

## Full-scale user inputs

For real operation, a user normally supplies:
- a local PGN corpus
- a list of Lichess study URLs or equivalent source manifests
- optional generated reports or study documents

## Document model

Markdown docs in the repo are the primary browser-readable document layer.
PDFs are optional derived exports when a workflow needs them.
