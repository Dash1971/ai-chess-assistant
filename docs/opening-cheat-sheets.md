# Opening Cheat Sheets

These are the canonical browser-readable opening cheat sheets for this repo.

- [Stonewall Playbook](stonewall-cheatsheet.md)
- [French Defense Playbook](french-cheatsheet.md)

## Canonical format rule

For the GitHub repo surface, the cheat sheets live in markdown first.

PDF generation remains useful for export and printing, but PDFs are **derived outputs**, not the canonical source documents.

## Provenance

The two cheat sheets are based on recurring patterns from Aman Hambleton's public speedrun study material:

- Stonewall material associated with `wonestall`
- French material associated with `sterkurstrakur`
